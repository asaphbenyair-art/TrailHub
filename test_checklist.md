# TrailHub — צ'קליסט בדיקות End-to-End (ידני)

מסמך זה מרכז את כל תרחישי הבדיקה הידניים ליצירת טיולים ולזרימות המטייל.
לכל זרימה מפורטים השלבים המדויקים לביצוע. סמן ✅ / ❌ לצד כל שלב.

**משתמשי בדיקה** (סיסמה `password123`):
`user@trailhub.co.il` (מטייל), `roei@trailhub.co.il` (מדריך), `coordinator@trailhub.co.il`, `emergency@trailhub.co.il`, `fieldguide@trailhub.co.il`

**הערה:** יצירת טיול (אשף מלא, GPX, מפה, נקודות עניין) — Web בלבד. עריכת שדות פשוטים — גם מובייל.

## זרימה 1 — טיול יום מודרך · בתשלום · ציבורי

### יצירה
1. התחבר כמדריך (`roei@trailhub.co.il`) → מצב מדריך → דשבורד.
2. לחץ "צור טיול חדש".
3. **שלב 1 (פרטים):** בחר סוג "טיול יום". מלא שם, תיאור, תאריך עתידי, שעת מפגש, שעת סיום משוערת, איזור, נקודת מפגש, העלה תמונה ראשית + 1-2 תמונות נוספות.
4. **שלב 2 (מסלול):** בחר סוג מסלול, העלה קובץ GPX → ודא שהמסלול מצויר על המפה ושגרף הגובה נוצר. עבור למצב "עריכה" והוסף 3-4 נקודות עצירה בלחיצה על המפה; ודא אזהרת "נקודה רחוקה מהמסלול" כשמניחים נקודה >10מ' מהמסלול. מלא אורך ומשך.
5. **שלב 3 (פרמטרים):** בחר רמת קושי, גיל מין/מקס, כושר, מקס/מין משתתפים, בחר לפחות פריט ציוד אחד, הוסף תגיות מאפיין, הגדר שדה הרשמה דינמי אחד (למשל checkbox הצהרת בריאות).
6. **שלב 4 (תשלום וביטול):** הגדר מחיר > 0, הגדר עד 3 מדרגות החזר.
7. **שלב 5 (פרסום):** עבור בין השלבים בלחיצה על שמות השלבים למעלה (ניווט חופשי) — ודא שאין ולידציה בעת מעבר שלב.

### שמירת טיוטא
8. בכל שלב לחץ "שמור טיוטא" — ודא שנשמר גם ללא מילוי כל שדות החובה, וללא ולידציה.
9. חזור לדשבורד → ודא שהטיוטא מופיעה עם תג "טיוטא".

### פרסום
10. חזור לאשף. בשלב 5 בחר "ציבורי" ולחץ "פרסם".
11. ודא ולידציית שדות חובה **מיידית** (<1 שנייה, ללא קריאת שרת): שדות חסרים מקבלים מסגרת אדומה + הודעה, טאב עם שגיאות מקבל תג אזהרה אדום עם מספר, גלילה אוטומטית לטאב ולשדה הראשון החסר, באנר עליון "נא למלא את כל שדות החובה לפני פרסום" עם ספירת החסרים.
12. השלם את החסר → פרסם → ודא שהטיול עובר לסטטוס "ציבורי" ומופיע בחיפוש.

### הרשמה כמטייל
13. התנתק, התחבר כמטייל (`user@trailhub.co.il`). חפש את הטיול → פתח עמוד טיול.
14. ודא סדר עמוד: תמונות → שם → מדריך+דירוג → תגיות → סטטיסטיקות → פס תפוסה → תאריך+שעה → נקודת מפגש → תיאור → מפה+נקודות → גרף גובה → ציוד → חומרי מקור → שו"ת → ביקורות → טרמפים → מדיניות ביטול.
15. לחץ "להרשמה" → **שלב 1:** אשר מדיניות ביטול (checkbox). **שלב 2:** מלא שדה דינמי. **שלב 3:** תשלום (Authorization). **שלב 4:** התראת סף ביטול. **שלב 5:** מסך אישור עם שם/תאריך/שעה/מדריך/מועד חיוב/הוסף ליומן Google.
16. ודא שהכרטיס בחיפוש ובעמוד מציג באנר ירוק "רשום לטיול ✓" + כפתור "בטל הרשמה".

### ביטול
17. לחץ "בטל הרשמה" → ודא דיאלוג ביטול חכם: סכום החזר מדויק (% לפי מדיניות מול הזמן) + הודעה מותאמת ("הטיול עדיין פתוח" / "נותרו X מקומות" / "מלא — X ברשימת המתנה") + כפתורים "בטל הרשמה" ו-"חזור".
18. לחץ "חזור" → נשאר רשום. לחץ שוב "בטל הרשמה" → אשר → ודא החזר וסטטוס חוזר ל"להרשמה".

### שאלות ותשובות
19. כמטייל רשום/מתעניין: לחץ מחוון שו"ת בכרטיס → נפתח מודל (לא עמוד). שאל שאלה גלויה ושאלה פרטית.
20. התחבר כמדריך → ודא שרואה את כל השאלות (כולל פרטית עם תג "פרטי"). ענה. סמן תשובה כרשמית.
21. חזור כמטייל → ודא מחוון "תשובה חדשה" כתום, ולאחר קריאה הופך אפור. הגב בתוך אותו שרשור (threaded).

### טרמפים
22. כמטייל רשום: לחץ מחוון "טרמפים" בכרטיס → נפתח מודל. הצע טרמפ (עיר, מקומות, כיוון). סמן "אני מחפש טרמפ".
23. ממשתמש רשום אחר: "תפוס מקום" → ודא שנפתח צ'אט פרטי. ודא שהטרמפ שלי מסומן "הטרמפ שלי".
24. בטל טרמפ → ודא שרק מי שתפס מקום מקבל התראה (לא כל הנרשמים).

### הודעות קבוצתיות
25. כמדריך: בכרטיס הדשבורד לחץ "הודעות קבוצתיות" → מודל. שלח הודעה חופשית.
26. כמטייל: ודא שההודעה מגיעה בהתראות, והלחיצה מנווטת לעמוד הטיול וגוללת למקטע ההודעות.

## זרימה 2 — טיול יום מודרך · חינם · ציבורי

### יצירה
1. כמדריך, צור טיול יום כמו בזרימה 1, אך **בשלב 4 הגדר מחיר = 0** (או השאר ריק ואשר את האזהרה → טיול חינם).

### שמירת טיוטא / פרסום
2. שמור טיוטא ופרסם כ"ציבורי" — זהה לזרימה 1.
3. ודא שהכרטיס והעמוד מציגים "חינם" בירוק (#3d8f5f) במקום "0₪".

### הרשמה כמטייל
4. לחץ "להרשמה" → ודא ש**אין** שלב מדיניות ביטול, **אין** checkbox "אני מסכים", **אין** שלב תשלום ולא התראת חיוב.
5. ודא מסך אישור ללא מועד חיוב/החזר.
6. ודא באנר "רשום לטיול ✓" + "בטל הרשמה".

### ביטול
7. לחץ "בטל הרשמה" → ודא הודעת ביטול פשוטה ("לבטל את ההרשמה?") **ללא** חישובי החזר. אשר → הוסר.

### שאלות ותשובות / טרמפים / הודעות קבוצתיות
8. זהה לזרימה 1 (שלבים 19-26).

## זרימה 3 — טיול יום מודרך · פרטי

### יצירה / פרסום
1. כמדריך, צור טיול יום (בתשלום או חינם). בשלב 5 בחר **"פרסום פרטי"** ופרסם.
2. ודא שהטיול **אינו** מופיע בחיפוש הציבורי.
3. העתק את הלינק הישיר (כפתור "העתק לינק" בדשבורד).

### הרשמה כמטייל
4. כמטייל, פתח את הלינק הישיר → ודא גישה מלאה: הרשמה, תשלום, רשימת המתנה, טרמפים — הכל עובד.
5. שתף את הלינק למשתמש אחר → ודא שגם הוא נכנס ונרשם דרך הלינק.

### ביטול / שו"ת / טרמפים / הודעות קבוצתיות
6. זהה לזרימות 1-2. ודא שכל התכונות פעילות גם בטיול פרטי.
7. ודא שמתעניינים מקבלים התראה אם הטיול משתנה מציבורי לפרטי.

## זרימה 4 — מסע (רב-יומי) עם GPX לכל יום

### יצירה
1. כמדריך, צור טיול חדש → **שלב 1: בחר סוג "מסע"**.
2. הגדר תאריך התחלה ותאריך סיום → ודא שהימים **נוצרים אוטומטית** לפי טווח התאריכים (אין כפתור "הוסף יום").
3. ודא שכל יום מוצג באקורדיון (מכווץ כברירת מחדל) עם: מספר יום, תאריך נגזר (התחלה + מספר יום), שעת מפגש.
4. פתח יום → מלא: שעת מפגש, שעת סיום משוערת, רמת קושי, שם ותיאור, ציוד נוסף ליום, **העלה GPX ליום זה** והוסף נקודות עצירה על המפה (אותה לוגיקה כמו טיול יום, אזהרת 10מ').
5. חזור על כך לכל יום. סמן יום אחד כ"יום מנוחה" (ללא GPX).
6. לחץ "הרחב הכל / צמצם הכל" → ודא פתיחה/כיווץ של כל הימים.
7. בחר אופן הרשמה (מסע מלא / ימים בודדים / גמישות), מחיר, מדיניות ביטול.

### שמירת טיוטא / פרסום
8. שמור טיוטא בכל שלב. פרסם כ"ציבורי".
9. ודא שהכרטיס מציג תג "מסע · X ימים" וטווח תאריכים.

### הרשמה כמטייל
10. פתח עמוד המסע → ודא ציר הזמן של הימים (אקורדיון, מכווץ כברירת מחדל, "הרחב הכל"), עם מפה/גובה/נקודות/ציוד/שעות לכל יום.
11. הירשם (לפי אופן ההרשמה שהוגדר). ודא באנר "רשום לטיול ✓" + "בטל הרשמה" בראש הכרטיס והעמוד — **זהה לטיול יום**.

### ביטול
12. לחץ "בטל הרשמה" → ודא דיאלוג ביטול חכם (החזר + הודעת הקשר). אשר.

### שאלות ותשובות
13. ודא שרשור שו"ת **נפרד לכל יום** במסע.

### טרמפים
14. ודא לוח טרמפים רק ליום הראשון (הגעה) וליום האחרון (חזרה) — לא לימי הביניים.

### הודעות קבוצתיות
15. כמדריך שלח הודעה קבוצתית → ודא שמגיעה לכל המחוברים למסע.

## זרימה 5 — טיול עצמאי · בתשלום · עם נקודות עניין ושמע

### יצירה
1. כמדריך, בחר יצירת **"טיול עצמאי"**.
2. **שלב 1:** שם, תיאור, איזור, תמונה ראשית (ללא תאריך/שעה).
3. **שלב 2:** העלה GPX. הוסף 6-8 נקודות עניין. לכל נקודה מלא: **הוראות ניווט צעד-אחר-צעד**, **חומר הדרכה (טקסט)**, **אזהרת בטיחות**, והעלה **קובץ שמע (הקלטת קול)** לחלק מהנקודות. הוסף PDF (עד 2) לחלק מהנקודות.
4. **שלב 3:** בחר ציוד, תגיות (כולל תגיות ייחודיות לעצמאי: מדורה/זריחה/עונתי), **הגדר חלון גישה** (למשל 30 יום), **מחיר > 0**.
5. **שלב 5:** בחר נראות חומרי מקור (תצוגה מוקדמת / רק בזמן הטיול). פרסם.

### שמירת טיוטא / פרסום
6. שמור טיוטא בכל שלב. ודא ולידציה מיידית של שדות חובה ייחודיים (הוראות ניווט, חומר הדרכה, חלון גישה, מחיר).
7. פרסם → ודא הופעה בטאב "עצמאיים" הנפרד בחיפוש (לא מעורב עם מודרכים).

### רכישה כמטייל (מקבילה ל"הרשמה")
8. כמטייל, פתח עמוד עצמאי → **לפני רכישה:** ודא מפה עם קו מסלול בלבד (ללא סמני נקודות), גרף גובה, ציוד; חומרי מקור ונקודות **חסומים**.
9. לחץ "רכוש" → תשלום מיידי וסופי (ללא Authorization/Capture, ללא מדיניות ביטול).
10. **אחרי רכישה:** ודא סמני נקודות ממוספרים, נקודת GPS כחולה, ניווט, TTS, נגני שמע לכל נקודה עם קול, חומרי מקור פתוחים.
11. בכרטיס "עצמאיים" ב"הטיולים שלי": ודא שני כפתורים — "▶ התחל / המשך" (מצב שטח) ו-"📖 למד על הטיול" (מצב עיון). ודא מחוון חלון גישה (ירוק/כתום/אדום/פג-תוקף).
12. בדוק מצב ניווט בשטח: "התחל טיול" → בדיקת קרבה 20מ' לנקודת התחלה → הגעה לנקודה (רטט/צליל/פתיחת תוכן/נגינת שמע אוטומטית) → "יצאתי לדרך" → זיהוי עזיבה ב-50מ'.
13. שתף גישה עד 3 אנשים (חיפוש לפי שם משתמש).

### ביטול / דיספיוט
14. ודא ש**אין** ביטול הרשמה רגיל (רכישה סופית). בדוק "בקשה להחזר" (פגם בתוכן / רכישה בטעות) → ודא שהתלונה נשלחת לאדמין ולמדריך.

### שאלות ותשובות / טרמפים
15. ודא ש**אין** שו"ת ו**אין** לוח טרמפים לטיול עצמאי (לא רלוונטי).

### ביקורות
16. ודא שכפתור "כתוב ביקורת" מופיע **רק** בטאב "עצמאיים", ונפתח (משך הטיול − 2) שעות אחרי הרכישה. לפני כן: "ביקורת תתאפשר בעוד X שעות".

### הודעות קבוצתיות
17. ודא ש**אין** הודעות קבוצתיות (אין ערוץ תקשורת בטיול עצמאי) — עדכוני תוכן מגיעים אוטומטית לרוכשים בתוך חלון הגישה.

## זרימה 6 — טיול עצמאי · חינם

### יצירה / פרסום
1. כמדריך, צור טיול עצמאי כמו בזרימה 5 אך **מחיר = 0**.
2. ודא תג "חינם" ומסנן "חינם בלבד" בחיפוש עצמאיים.

### רכישה / הצטרפות כמטייל
3. לחץ "הצטרף" → ודא ש**אין** תשלום כלל.
4. ודא **גישה מלאה וקבועה** (ללא חלון גישה, ללא פג-תוקף): נקודות, שמע, חומרי מקור פתוחים תמיד.

### ניווט / שיתוף / ביקורות
5. זהה לזרימה 5 (שלבים 11-13, 16) — כולל מצב שטח, TTS/שמע, שיתוף עד 3, וביקורת מטאב "עצמאיים".

### ללא: ביטול-החזר / שו"ת / טרמפים / הודעות קבוצתיות
6. ודא שאין זרימות תשלום/החזר, אין שו"ת, אין טרמפים, אין הודעות קבוצתיות.

## בדיקות רוחב (חוצות-זרימות)
- **שפה:** החלף עב/EN → ודא שכל ה-UI מתורגם (כולל אשף היצירה, כרטיסים, סטטוסים, איזורים) והפריסה מתהפכת RTL↔LTR. תוכן משתמש נשאר בשפת המקור עם כפתור Translate.
- **לוח שנה עברי:** החלף ע/ל → ודא המרת כל התאריכים בכל המסכים.
- **מצב כהה/בהיר:** ודא ששני המצבים נראים תקין.
- **התראות:** ודא זמן-אמת (ללא רענון), מחוון אדום עם ספירה, סימון כנקרא ללא ניווט, ו-deep-links מדויקים.
- **כרטיס טיול אחיד:** ודא שהכרטיס נראה זהה בחיפוש, "הטיולים שלי", ודשבורד המדריך.
