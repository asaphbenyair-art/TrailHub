# Wireframes

קבצי HTML סטטיים של מסכי הליבה של TrailHub. פתח ישירות בדפדפן — אין צורך בשרת.

| קובץ | מסך |
|------|-----|
| `01_trip_card.html` | כרטיס טיול — שלושה מצבים (פתוח / מלא / waitlist) |
| `02_trip_detail.html` | דף טיול מלא |
| `03_registration_flow.html` | זרימת הרשמה (רגילה / על תנאי / waitlist) |
| `04_search_page.html` | דף חיפוש וסינון |
| `05_my_trips.html` | הטיולים שלי (משתמש) |
| `06_guide_profile.html` | פרופיל מדריך |
| `07_guide_dashboard.html` | דשבורד מדריך |
| `08_trip_wizard.html` | אשף יצירת טיול |
